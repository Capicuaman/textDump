## Quick Start

1. Save the setup script and run it with `sudo`
2. Set a parent password when prompted
3. Log out and select "i3" session
4. Log in as `kiduser`

The launcher appears automatically with big, friendly buttons. Children press Super+Home anytime to return to the launcher, and you click the lock icon with your password to exit.
