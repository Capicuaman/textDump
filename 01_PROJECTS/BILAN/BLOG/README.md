The published and ready-to-publish blog articles for the BILAN brand, along with the Jekyll static site infrastructure.

## Contents

**Markdown:** kramdown with GitHub Flavored Markdown (GFM)
**Syntax Highlighting:** Rouge
**Location:** `jekyll/carte-noire-gh-pages/`
