# TikTok Testing Campaign - January 2025

## Quick Start

Generated on: January 23, 2025

This directory contains your TikTok engagement testing materials:

1. **2025-01-23-batch.md** - 10 complete video scripts with 3 hook variations each
2. **engagement-tracking-2025-01.md** - Tracking spreadsheet for metrics and analysis

## What's Included

### 10 Test Videos
- 2 Educational (Sodium science, 1000mg explained)
- 2 Myth-busting (Clear urine, 8 glasses myths)
- 2 Before/After (Workout transformation, Keto solution)
- 2 Quick Tips (Urine color chart, Pre-workout ritual)
- 2 Trending Formats (Hydration hack, GRWM gym edition)

### Testing Variables
- Hook styles (Question, Statement, Shock Value) - 3 per video = 30 total hooks
- Video lengths (15s, 30s, 45s, 60s)
- Tones (Educational, Humorous, Inspirational, Serious)
- Formats (Talking Head, B-roll, Animation, Demonstration)
- CTAs (Comment, Follow, Link, Save)

## How to Use

1. **Review scripts** in 2025-01-23-batch.md
2. **Choose hooks** - Pick A, B, or C for each video
3. **Produce videos** - Create content using scripts
4. **Post 2-3x per week** - Spread out for best testing
5. **Track metrics** - Fill in engagement-tracking-2025-01.md after 24h, 48h, 7d
6. **Analyze weekly** - Review patterns on Sundays
7. **Repurpose winners** - Adapt top performers for other platforms
