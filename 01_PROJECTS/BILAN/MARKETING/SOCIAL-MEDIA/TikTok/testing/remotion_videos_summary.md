## Rendering Process

### Tools Used
1. **Remotion** - React-based video generation framework
2. **Node.js** - Runtime environment
3. **FFmpeg** - Video encoding
4. **Chrome Headless Shell** - Browser rendering

### Command Used
```bash
CSV_FILE=src/data/videos-mx-2026-01-23.csv npm run render
```

### CSV Structure
- 16 columns per row
- Headers: id, type, title, hook, mainPoints, conclusion, myth, truth, explanation, tip, reason, duration, scenes, trendingFormat, cta, brandColor
- 10 data rows (one per video)
- Pipe-separated (|) values for multi-point fields
