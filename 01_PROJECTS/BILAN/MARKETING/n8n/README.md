# n8n - Social Media Content Automation

This directory contains n8n workflows and documentation for **social media marketing automation**.

## Purpose

Automated social media content generation, scheduling, and publishing workflows using n8n and AI tools (primarily Claude API).

## Contents

- `SM-Posts/` - Social media post automation workflows
  - AI content generation
  - Theme selectors
  - Content formatting and quality control
  - Google Docs integration
- `bateria.md` - Content batching strategy

## Key Features

- Automated social media post generation
- Theme-based content creation
- Multi-platform formatting (Instagram, TikTok, WhatsApp)
- AI-powered content quality control
