# BILAN Mexican States - Still Image Generation Prompts

## Overview

This document contains still image generation prompts for all 32 Mexican states. These prompts are optimized for AI image generation platforms (Midjourney, DALL-E 3, Stable Diffusion, Adobe Firefly, or NanoBanana) and create static product photography with state-specific backgrounds.

**Format:** 9:16 vertical (1024x1820 or 1080x1920)
**Style:** Photorealistic product photography
**Purpose:** Social media posts, paid ads, website banners

---

## Master Prompt Template Structure

Each prompt follows this structure:
1. **Main Prompt:** Detailed scene description
2. **Negative Prompt:** Elements to avoid
3. **Technical Settings:** Resolution, aspect ratio, style parameters
4. **State Context:** Why this landmark matters

## Usage Instructions

### For Midjourney

1. Copy main prompt
2. Add at end: `--ar 9:16 --style raw --v 6`
3. Use `--no [negative prompt items]` for exclusions

**Example:**
```
[Main prompt text] --ar 9:16 --style raw --v 6 --no cartoon, illustration, blurry
```

### For DALL-E 3

1. Copy main prompt (max 400 characters, condense if needed)
2. Specify: "Create a vertical 9:16 image"
3. DALL-E doesn't support negative prompts directly; focus on positive descriptors

### For Stable Diffusion / Adobe Firefly

1. Use main prompt in positive field
2. Use negative prompt in negative field
3. Set aspect ratio to 9:16 (1024x1820 or custom)
4. CFG Scale: 7-8
5. Steps: 30-50
6. Sampler: DPM++ 2M Karras or Euler A

### For NanoBanana (Google Imagen)

1. Format as JSON with subject, environment, composition, product_details
2. Specify 9:16 aspect ratio in config
3. Use natural language description
