### How to Update Files

**Adding a New Track:**
1. Add entry to `tracks_database.json`
2. Add to appropriate section in `BEST_MINIMAL_TECHNO_ALL_TIME.md`
3. Update `QUICK_REFERENCE.md` if essential
4. Update line count in this INDEX

1. Research thoroughly (3+ sources)

4. Add key tracks to database

**Creating a New Set:**
1. Follow template in `minimal_set_structure.md`
2. Select 12-20 tracks from database
3. Verify BPM progression
4. Check key compatibility
5. Document energy levels
6. Create new .md file if different style
