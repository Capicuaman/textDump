## üìä MINIMAL TECHNO SPECS AT A GLANCE

### Standard Parameters

```
BPM:        120-132 (typical: 126-128)
Keys:       Em, Am, Dm, Cm, Gm, Fm, Bm (minor = 99%)
Duration:   6:30 - 8:45 minutes (typical: 7:30)
Time Sig:   4/4 (always)
Kick:       4-on-the-floor (steady)
Bassline:   Hypnotic, minimal melodic
Drums:      Minimal percussion, hi-hat patterns
Vibe:       Hypnotic, repetitive, groove-based
```

### Key Compatibility Quick Reference

**Most Compatible Transitions:**
- Em ‚Üí Am (great pairing)
- Am ‚Üí Dm (smooth)
- Dm ‚Üí Gm (natural)
- Cm ‚Üí Gm (good)
- Fm ‚Üí Bm (works well)

**Pitch Shift Hack:**
- Use DJ software to shift key ¬±3-6 semitones
- Allows mixing any key together
- Works well for minimal (less melodic interference)

## üéõÔ∏è MIXING ESSENTIALS

### Critical Mixing Techniques

| Technique | How | When | Why |
|-----------|-----|------|-----|
| **Beatmatch** | Sync BPMs or manual | Every transition | Essential foundation |
| **Long Blend** | 8-16 bar crossfade | Default | Hypnotic flow |
| **EQ Filter** | High-pass strip out | Peak transitions | Smooth element layering |
| **Cue Points** | Mark key sections | Pre-set in software | Fast navigation |
| **Loop Extend** | Extend intro/outro | Bridge gaps | Smooth timing |

### Energy Management Rules

- ‚úÖ Increase energy every 5-8 tracks
- ‚úÖ Use breaks every 4-6 tracks
- ‚úÖ Peak at 55-70% through set
- ‚úÖ Never spike energy 20%+ at once
- ‚ùå Don't plateau too long
- ‚ùå Avoid late peaks (diminishing returns)

## üìö KEY REFERENCES (ONE-LINERS)

### The Most Important Concepts

**"Minimal is About Repetition"**
- Don't change elements every 8 bars
- Let one groove sit for 2-3 minutes
- Listener goes into trance after 2+ minutes

**"Less is More"**
- Start with 2-3 elements (kick, hat, bass)
- Add ONE element per 4-8 bars
- Build from nothing to everything

**"The Kick is Everything"**
- 4-on-the-floor is sacred
- Slight variations in timing create groove
- Kick defines the whole vibe

**"BPM Consistency is Key"**
- Minimal thrives on steady groove
- Don't jump 10+ BPM mid-set
- Gradual 2-3 BPM shifts acceptable

**"Hypnosis > Excitement"**
- Goal: put dancers in trance state
- Not about "bangers" or energy spikes
- Subtle variations matter most

## ‚ùì FAQ (Quick Answers)

**Q: What BPM should I start at?**  
A: 124-126 BPM. Allows for gradual tempo progression.

**Q: Should I focus on keys?**  
A: Yes, but minimal is forgiving. Minor keys (Em, Am, Dm) dominate.

**Q: How long should transitions be?**  
A: 8-16 bars (minimal likes long blends). ~30-60 seconds typical.

**Q: What if I don't have DJ software?**  
A: Start with YouTube mixes + Spotify for learning.

**Q: Where do I find rare/classic vinyl?**  
A: Discogs, Discrepancies, local record shops.

**Q: Can I mix different keys?**  
A: Yes, use pitch shift (¬±6 semitones max recommended).

**Q: What's the best controller?**  
A: Pioneer DDJ-1000 or Technics SL-1200 turntables (pro).

**Q: How do I develop my style?**  
A: Mix the essentials first, then add your own discovery.
