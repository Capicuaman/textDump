This project aims to compile **the definitive collection of minimal techno music** - from legendary pioneers of the 1990s to modern masters. The focus is on curating, organizing, and preparing a professional DJ set using the best tracks in the genre.

### Project Goals

1. **Research & Discovery** - Identify essential minimal techno tracks across all eras
2. **Curation** - Select the absolute best tracks for a professional set
3. **Organization** - Create structured databases for easy access and mixing
4. **Set Building** - Develop complete DJ sets at different duration/energy levels
5. **Download Management** - Provide links and resources for acquiring tracks

## üìù NOTES & CONVENTIONS

### BPM Ranges
- **120-124 BPM:** Intro/Build/Resolution
- **125-128 BPM:** Development/Groove
- **129-132 BPM:** Peak energy

### Energy Levels
- **0-40%:** Intro/Wind-down
- **40-60%:** Development
- **60-80%:** Build/Groove
- **80-100%:** Peak moments

### Key Compatibility (Circle of Fifths)
- Most compatible: 0-2 semitones difference
- Acceptable: 1-5 semitones with caution
- Use pitch shift for non-compatible keys
