# Gemini CLI Shortcut Keys

This document lists the keyboard shortcuts for navigating and interacting with the Gemini CLI.

## General Navigation & Control

| Shortcut | Description |
|---|---|
| `Esc` | Closes dialogs and suggestions. |
| `Ctrl + C` | Cancels the ongoing request and clears the input. Pressing it twice exits the application. |
| `Ctrl + D` | Exits the application if the input is empty. Press twice to confirm. |
| `Ctrl + L` | Clears the screen. |
| `Ctrl + S` | Allows long responses to print fully, disabling truncation, and also toggles copy mode. |
| `Ctrl + T` | Toggles the display of tool descriptions. |
| `Ctrl + Y` | Toggles auto-approval (YOLO mode) for all tool calls. |
| `Ctrl + F` | Toggles focus between the input and interactive shell if active. |

## Input Prompt Shortcuts

| Shortcut | Description |
|---|---|
| `!` | Toggles shell mode when the input is empty. |
| `Enter` | Submits the current prompt. |
| `Up/Down Arrows` | Navigate through input history. |
| `Tab` | Autocompletes the current suggestion if one exists. |
| `Ctrl + X` | Opens the current prompt in your default system editor. |
| `Ctrl + Z` | Undoes the last action in the input prompt. |
| `Ctrl + Shift + Z` | Redoes the last undone action in the input prompt. |
| `Meta + Delete` / `Ctrl + Delete` | Deletes the word to the right of the cursor. |
| `Meta + B` (Mac) | Moves back one word. |
| `Meta + F` (Mac) | Moves forward one word. |
| `Meta + M` (Mac) | Toggles markup view. |

## Radio Button Select

| Shortcut | Description |
|---|---|
| `Up/Down Arrows` | Move selection up or down. |
| `Enter` | Confirms selection. |
| Numbers | Select an item by its number. For items greater than 9, press digits in quick succession. |
