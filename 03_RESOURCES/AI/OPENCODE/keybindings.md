# OpenCode Key Bindings

This is a list of the default key bindings in OpenCode, based on the configuration options.

## Leader Key
- Leader key: `ctrl+x` (used as prefix for many commands)

## Application
- Exit: `ctrl+c`, `ctrl+d`, `<leader>q`

## Editor
- Open editor: `<leader>e`

## UI
- Theme list: `<leader>t`
- Sidebar toggle: `<leader>b`
- Status view: `<leader>s`

## Sessions
- Export session: `<leader>x`
- New session: `<leader>n`
- Session list: `<leader>l`
- Session timeline: `<leader>g`
- Share session: `none` (disabled by default)
- Unshare session: `none` (disabled by default)
- Interrupt session: `escape`
- Compact session: `<leader>c`
- Cycle child session: `ctrl+right`
- Cycle child session reverse: `ctrl+left`

## Messages
- Page up: `pageup`
- Page down: `pagedown`
- Half page up: `ctrl+alt+u`
- Half page down: `ctrl+alt+d`
- First message: `ctrl+g`, `home`
- Last message: `ctrl+alt+g`, `end`
- Copy message: `<leader>y`
- Undo: `<leader>u`
- Redo: `<leader>r`
- Toggle conceal: `<leader>h`

## Models
- Model list: `<leader>m`
- Cycle recent model: `f2`
- Cycle recent model reverse: `shift+f2`

## Commands and Agents
- Command list: `ctrl+p`
- Agent list: `<leader>a`
- Cycle agent: `tab`
- Cycle agent reverse: `shift+tab`

## Input
- Clear input: `ctrl+c`
- Forward delete: `ctrl+d`
- Paste: `ctrl+v`
- Submit: `enter`
- Newline: `shift+enter`, `ctrl+j`
- History previous: `up`
- History next: `down`

Note: These are the default bindings. You can customize them in your `opencode.json` config file. Set a binding to `"none"` to disable it.
