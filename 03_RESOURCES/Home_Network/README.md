Reference materials and documentation for home network setup, configuration, and management.

## Directory Structure

- Routers, switches, access points
- Network attached storage (NAS)
- Cables, adapters, and physical infrastructure

Visual representations of network topology:
- Physical network layout

- VLAN configurations
- IP address schemes

### Configuration/
Configuration files, settings, and setup guides:
- Router/firewall configurations
- DNS settings
- DHCP configurations
- Port forwarding rules
- VPN configurations

### Security/
Network security documentation:
- Firewall rules
- Security best practices
- Vulnerability assessments
- Access control policies
- Network monitoring

### Troubleshooting/
Common issues and solutions:
- Connectivity problems
- Performance issues
- Hardware failures
- Configuration errors
- Diagnostic commands and tools

### Documentation/

## Quick Reference

### Common Tools
- `ip addr` / `ifconfig` - View network interfaces
- `ping` - Test connectivity
- `traceroute` / `traceroute6` - Trace network path
- `nmap` - Network scanning
- `netstat` / `ss` - Network statistics
- `dig` / `nslookup` - DNS queries

### Useful Commands
- `arp -a` - View ARP cache
- `route -n` - View routing table
- `sudo tcpdump` - Packet capture
- `iftop` - Network bandwidth monitoring
