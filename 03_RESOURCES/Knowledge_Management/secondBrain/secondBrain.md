# OpenCode Second Brain Overview

## Text Organization & Management

### Directory Structure Setup
```bash
opencode "Create a structured directory system for my knowledge base with folders for: projects, ideas, references, and daily notes"
```

### Template Generation
```bash
opencode "Generate templates for different types of notes: meeting notes, book summaries, project plans, and daily reflections"
```

## Content Creation & Enhancement

### Content Transformation
```bash
opencode "Transform these bullet points into a well-structured article about [topic]"
```

### Insight Extraction
```bash
opencode "Extract key insights from this article and create a summary with actionable points"
```

### Knowledge Connection
```bash
opencode "Create connections between my notes on [topic1] and [topic2] - find common themes"
```

## Knowledge Retrieval & Search

### Topic-Based Search
```bash
opencode "Search through all my .md files and find everything related to 'productivity systems'"
```

### Pattern Analysis
```bash
opencode "Read my last 5 daily notes and identify recurring themes or patterns"
```

## Automated Workflows

### Daily Note Creation

```bash
opencode "Create a daily note template with date, weather, goals, and reflection sections"
```

### Weekly Review Generation
```bash
opencode "Generate a weekly review by compiling all my daily notes from this week"
```

## Content Analysis

### Writing Improvement

## Key Principle

The key is using natural language commands to organize, search, and synthesize your text-based knowledge base without leaving the terminal.
