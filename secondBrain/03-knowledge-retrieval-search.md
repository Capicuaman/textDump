## Best Practices for Knowledge Retrieval

### Search Strategy
- Use specific, targeted queries
- Combine multiple search terms
- Use quotes for exact phrases
- Leverage metadata and tags
- Refine searches based on results

### Result Evaluation
- Verify relevance of results
- Check context and accuracy
- Cross-reference important findings
- Update outdated information
- Document new insights

### Knowledge Maintenance
- Regularly review and update search results
- Archive outdated information
- Consolidate duplicate content
- Improve tagging and categorization
- Maintain search query library

This comprehensive search and retrieval system transforms OpenCode into a powerful knowledge discovery tool that helps you efficiently navigate and leverage your accumulated knowledge.
